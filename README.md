# 📸 Photo Gallery

A simple photo gallery web application built with Python and HTML.

## 🚀 About the Project

This repository contains a photo gallery web app where you can display images in a web browser.  
It’s created with Python for the backend logic and HTML/CSS for the frontend structure and design.

## 🛠 Technologies Used

- Python  
- HTML5  
- CSS3  
- (Optional: Flask/Django — if you used one)

## 📸 Features

- Displays images in a gallery layout  
- Easy to browse interface  
- Clean frontend and simple backend  

## 🧠 How to Run Locally

1. Make sure you have Python installed  
2. Navigate to the project folder  
3. Open terminal / command prompt  
4. Run the app:

bash
python app.py
